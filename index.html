Introduction to CSS Animations

What Are Animations?

Animations are visual changes that make elements move or change appearance over time. They bring websites to life by adding movement and interaction.

Keyframes

@keyframes is a CSS rule that lets you create animations by defining what styles an element should have at different points during the animation (for example: start, middle, end).

Basic Syntax

animation-name

Give your keyframe any name you want (like "slide", "fade", or "spin"). Once it's defined, you can reuse the animation as many times as you want. 

@keyframes fade {
  from { opacity: 0; }
  to { opacity: 1; }
}
from & to

Define where your animation starts (from) and where it ends (to). 

@keyframes fade {
  from { opacity: 0; }
  to { opacity: 1; }
}
percentages

Use percentages to add middle points to your animations. 0% = start, 50% = halfway, 100% = end. 

@keyframes bounce {
  0% { top: 0; }
  50% { top: 100px; }
  100% { top: 0; }
}
animation

Use the animation property to apply your keyframe to any element. The same keyframe can be reused on multiple elements! 

.box {
  animation: fade 2s ease-in-out;
}

.circle {
  animation: fade 3s linear;
}

Breaking Down the Code

@keyframes slide

Creates an animation called "slide" that can be reused on any element in your CSS.

from { left: 0; }
to { left: calc(100% - 50px); }

Defines the start position (left edge) and end position (right side) of the animation.

animation: slide 2s infinite

Applies the animation to an element, running for 2 seconds and repeating forever.
